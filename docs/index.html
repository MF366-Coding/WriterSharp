Why use WriterSharp?

Open-Source

Unlike most code editors out there, WriterSharp is fully open-source and licensed under the permissive MIT License.

Privacy-focused

WriterSharp is fully telemetry-free, being fully private.

Cross-platform

WriterSharp is cross-platform, running on Windows, Linux and MacOS.*

*WriterSharp on MacOS is currently untested.

Extensible

WriterSharp's plugin API is designed to learn from the mistakes of most text editors, making WriterSharp extremely extensible.
